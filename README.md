# DataWake 对外资料中心

本仓库集中管理 DataWake 面向客户、合作伙伴及其他外部对象公开分享的业务介绍、产品材料、PPT、HTML 和 PDF。

- [对外资料中心](https://datawake-ai.github.io/public-materials/)
- [Wake AI 业务介绍 V2.4（HTML）](https://datawake-ai.github.io/public-materials/materials/wake-ai-business-introduction/v2.4/)
- [Wake AI 业务介绍 V2.4（PDF）](https://datawake-ai.github.io/public-materials/materials/wake-ai-business-introduction/v2.4/Wake-AI-Business-Introduction-V2.4.pdf)
- [AI广告工作流实战：从一句话取数，到自动托管增长](https://rememberzx.github.io/ai-ads-workflow-presentation/)

## 目录约定

每套资料统一放在 `materials/<资料名称>/<版本>/` 下：

- `index.html`：在线浏览版本
- `*.pdf`：可下载、可转发版本
- `assets/`：HTML 使用的图片等静态资源

新版本保留独立目录，避免历史分享链接失效。

已经由独立公开仓库发布且存在外部分享链接的资料，可以先在资料中心登记入口；迁移前保留原地址，避免历史链接失效。
