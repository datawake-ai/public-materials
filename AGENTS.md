# 公开资料仓库操作规则

- 本仓库全部内容对互联网公开，任何提交都必须按公开材料处理。
- 未得到用户明确发布指令时，禁止从私有资料库复制或发布内容。
- 禁止提交客户定制报价、内部定价、成本、折扣权限、内部讲稿、未授权案例、凭证和未脱敏数据。
- 发布前检查 HTML 隐藏内容、JavaScript 数据、图片、文件元数据和 Git 历史。
- 每套自托管资料使用 `materials/<资料名>/<版本>/`，保留历史版本和旧链接。
- 新增资料时同步更新 `README.md` 和首页 `index.html`。
- 已由独立公开仓库发布的资料可以登记外部链接；迁移前不得破坏原公开地址。
- 提交信息、Issue 和 PR 内容使用中文。
